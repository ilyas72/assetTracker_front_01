# Project Title

Asset Tracker (Frontend)


## Installing

```
 Step 1 -> Go to the project directory in the terminal
 Step 2 -> Key in 'npm install --save'
 Step 3 -> Key in 'ng serve -o'
 Step 4 -> On browser tab http://localhost:4200
```
